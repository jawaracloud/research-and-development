# 99 — The Future of NATS

> **Type:** Explanation  
> **Phase:** Advanced & Real-World

## Overview

NATS is evolving rapidly. This lesson covers where the project is heading and the emerging features in the NATS ecosystem.

## 1. NATS Core Performance
- **Zero-Copy Improvements:** Ongoing work to reduce memory allocations in the hot path.
- **Protocol V2:** Efforts to modernize the underlying text protocol for even better efficiency.

## 2. Advanced JetStream Features
- **Deterministic Sequencing:** More precise control over how messages are indexed.
- **Extended Mirroring:** Even simpler ways to aggregate massive datasets across global clusters.

## 3. Tooling & Ecosystem
- **NATS Micro Evolution:** Adding more middleware and visualization directly into the micro-services framework.
- **Operator Improvements:** Making it easier to manage 1,000s of accounts and users in a GitOps way.

## 4. Connectivity Beyond the Cloud
- **WebAssembly (Wasm):** Running NATS clients inside Wasm runtimes at the extreme edge.
- **Embedded NATS:** Deeper integration into edge OSs and container runtimes.

## 5. The NATS Philosophy
NATS continues to double-down on its core promise: **"Connect Everything"**. 
It's not just a message broker; it's a "Connectivity Provider" that aims to simplify how we build distributed systems by removing the need for Load Balancers, Service Registries, and complex Sidecars.
