# 99 — Publishing Wasm Libraries to crates.io

> **Type:** How-To + Reference

## What makes a good Wasm library?

A Wasm library targets `wasm32-unknown-unknown` and exposes an API via `wasm-bindgen`. To be publishable:
1. Works on both native Rust (for testing) and Wasm (for browser use).
2. Has clear documentation with examples.
3. Provides TypeScript type definitions.
4. Is properly versioned via SemVer.

## Library structure

```
my-wasm-lib/
├── Cargo.toml
├── README.md
├── src/
│   ├── lib.rs         # Public API
│   ├── core.rs        # Implementation (no wasm-bindgen)
│   └── wasm.rs        # wasm-bindgen exports
├── tests/
│   ├── native_tests.rs # cargo test (fast)
│   └── wasm_tests.rs   # wasm-bindgen-test (browser)
└── pkg/               # Generated by wasm-pack
```

## Cargo.toml for a publishable library

```toml
[package]
name = "my-wasm-lib"
version = "0.1.0"
edition = "2021"
description = "A Wasm library for doing X in the browser"
license = "MIT OR Apache-2.0"
repository = "https://github.com/yourusername/my-wasm-lib"
homepage = "https://github.com/yourusername/my-wasm-lib"
documentation = "https://docs.rs/my-wasm-lib"
keywords = ["wasm", "webassembly", "browser"]
categories = ["wasm", "web-programming"]
readme = "README.md"

[lib]
crate-type = ["cdylib", "rlib"]  # cdylib for Wasm, rlib for native tests

[dependencies]
wasm-bindgen = { version = "0.2", optional = true }
serde = { version = "1", features = ["derive"] }
js-sys = { version = "0.3", optional = true }

[features]
default = []
wasm = ["wasm-bindgen", "js-sys"]

[dev-dependencies]
wasm-bindgen-test = "0.3"
```

## Separating platform concerns

`src/core.rs` — pure Rust, no Wasm dependencies:
```rust
pub fn fibonacci(n: u64) -> u64 {
    if n <= 1 { return n; }
    let mut a = 0u64;
    let mut b = 1u64;
    for _ in 2..=n {
        (a, b) = (b, a + b);
    }
    b
}

pub fn is_prime(n: u64) -> bool {
    if n < 2 { return false; }
    if n == 2 { return true; }
    if n % 2 == 0 { return false; }
    let mut i = 3u64;
    while i * i <= n {
        if n % i == 0 { return false; }
        i += 2;
    }
    true
}
```

`src/wasm.rs` — wasm-bindgen bindings:
```rust
#[cfg(feature = "wasm")]
mod wasm_api {
    use wasm_bindgen::prelude::*;
    use crate::core;

    #[wasm_bindgen]
    pub fn fibonacci(n: u64) -> u64 {
        core::fibonacci(n)
    }

    #[wasm_bindgen]
    pub fn is_prime(n: u64) -> bool {
        core::is_prime(n)
    }

    #[wasm_bindgen]
    pub fn primes_up_to(limit: u64) -> Vec<u64> {
        (2..=limit).filter(|&n| core::is_prime(n)).collect()
    }
}
```

`src/lib.rs`:
```rust
pub mod core;

#[cfg(feature = "wasm")]
pub mod wasm;
```

## Native tests

```rust
// tests/native_tests.rs
use my_wasm_lib::core;

#[test]
fn test_fibonacci() {
    assert_eq!(core::fibonacci(0), 0);
    assert_eq!(core::fibonacci(1), 1);
    assert_eq!(core::fibonacci(10), 55);
    assert_eq!(core::fibonacci(20), 6765);
}

#[test]
fn test_prime() {
    assert!(!core::is_prime(0));
    assert!(!core::is_prime(1));
    assert!(core::is_prime(2));
    assert!(core::is_prime(97));
    assert!(!core::is_prime(100));
}
```

Run: `cargo test`

## Wasm browser tests

```rust
// tests/wasm_tests.rs
use wasm_bindgen_test::*;
use my_wasm_lib::wasm_api;
wasm_bindgen_test_configure!(run_in_browser);

#[wasm_bindgen_test]
fn test_wasm_fibonacci() {
    assert_eq!(wasm_api::fibonacci(10), 55);
}
```

Run: `wasm-pack test --headless --chrome`

## Building and publishing

```bash
# Build the npm package (creates pkg/ directory)
wasm-pack build --target web

# Inspect generated types
cat pkg/my_wasm_lib.d.ts

# Publish to npm (JavaScript ecosystem)
wasm-pack publish

# Publish to crates.io (Rust ecosystem)
cargo publish
```
