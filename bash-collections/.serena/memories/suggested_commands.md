# Suggested Commands
General guidelines to use any script:
1. Navigate to the script directory (`cd <script-directory>`)
2. Read the `README.md` for specific requirements
3. Make the script executable: `chmod +x script.sh`
4. Run the script: `./script.sh`

System utils available: standard Linux utilities (`find`, `grep`, `awk`, `sed`), `docker`, `kubectl`, `aws`.
