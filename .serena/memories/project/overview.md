# Jawaracloud R&D Repository

Mono-repo for cloud-native projects and case studies. Follows the **Diátaxis** documentation framework.

## Core Tech Stacks
- **Go**: Microservices, K8s operators, eBPF, NATS.
- **Rust/Wasm**: WebAssembly, Web3.
- **Bash**: DevOps, Infrastructure automation.
- **TypeScript**: Playwright UI automation.
- **Solidity**: Smart contracts (Web3).
- **Infrastructure**: Kubernetes, Docker Compose, AWS.
